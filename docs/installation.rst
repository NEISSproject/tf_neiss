============
Installation
============


Stable release
--------------

To install tf_neiss, run this command in your terminal:

.. code-block:: console

    $ pip install tf_neiss

This is the preferred method to install tf_neiss, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for tf_neiss can be downloaded from the `remote repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/dboeckenhoff/tf_neiss

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/dboeckenhoff/tf_neiss/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _remote repo: https://github.com/dboeckenhoff/tf_neiss
.. _tarball: https://github.com/dboeckenhoff/tf_neiss/tarball/master
